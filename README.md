# 🍉 合成大西瓜 H5

一个可爱的水果合成游戏，支持手机和电脑游玩。

| 平台 | 链接 |
|------|------|

| **海外（GitHub）** | https://gaoyingxie.github.io/watermelon-game/ |

## ✨ 游戏特色

- 🍇 **11种水果**：从蓝莓到西瓜，越合越大
- 📱 **手机晃动**：晃动手机触发物理冲击，弹飞水果
- 🎨 **独特纹理**：每种水果都有自己的纹理图案
- ⚡ **连击系统**：多颗水果同时合成有额外奖励
- 🎯 **双平台部署**：国内访问网易云，海外访问 GitHub

## 🕹️ 操作说明

| 操作 | 说明 |
|------|------|
| 点击/触摸 | 放置水果 |
| 晃动手机 | 弹飞所有水果（5秒CD）|
| 相同水果碰撞 | 合成更大的水果 |

## 📝 游戏规则

- 点击屏幕放置水果
- 相同水果接触会合成更高级的水果
- 水果堆到顶部警戒线会游戏结束
- 尽可能合成大西瓜获得高分！
